### Self-Hosted Compliance

When you self-host the control plane, compliance becomes simpler:

| Benefit | How It Works |
|---------|-------------|
| Inherit your certifications | Your SOC 2, ISO 27001, HIPAA automatically cover Tero |
| Your compliance boundary | All data stays in your infrastructure |
| Your audit scope | Include Tero in your existing audits |
| Your data residency | Deploy in any region |
| Your controls | Use your existing security and compliance controls |

### Security Controls We Implement

| Control Area | What We Do |
|-------------|-----------|
| Access management | Role-based access, MFA required, least privilege |
| Data protection | Encryption at rest and in transit, secure deletion, documented retention |
| Change management | Version control, peer review, staging environment, rollback procedures |
| Incident response | Documented plan, prompt notification, post-incident analysis |
| Vendor management | Security assessments for critical vendors, data processing agreements |
| Security monitoring | Automated alerts, audit logging, vulnerability scanning |
| Employee security | Background checks where required, security training |
| Business continuity | Daily backups, disaster recovery procedures |
